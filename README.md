# Hunger Games Simulator

A real-time, browser-based simulation where tributes compete in a dynamic arena. 

## Features
- **Tribute AI:** Complex state machines for combat, alliance formation, and betrayal.
- **Nightly Recap System:** The game pauses at night to show a "Nightly Recap" of events.
- **Cinematic Replay Reel:** View actual 3-second animated clips of kills and alliances, captured in real-time.
- **Dynamic Visuals:** Custom canvas rendering with dynamic lighting, gore effects, and portrait-based memory tracking.

## Deployment (GitHub Pages)
This project is configured to deploy via GitHub Pages:
1. Navigate to your repository on GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment** > **Source**, ensure "Deploy from a branch" is selected.
4. Set the branch to `master` (or `main`) and the folder to `/(root)`.
5. Click **Save**. The simulator will be live at `https://[your-username].github.io/hunger-games-sim/`.
